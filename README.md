# Send Control

Send Control is a macOS menu bar utility that remaps Return and Shift+Return at the event-tap layer.

## Status

- Technical preview / beta
- Not notarized
- Intended for testers and advanced macOS users
- The distributable app package is published in this repository's GitHub Releases

## Where To Start

- If you want the packaged app, use the GitHub Releases page for this repository.
- The source tree and project files currently live under `Customize Return Key/`.
- This repository includes technical preview documentation in both English and Japanese.
