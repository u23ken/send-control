# Send Control クイックスタート

## 先に知っておくこと

- `アクセシビリティ` と `入力監視` の両方の権限が必要です。
- `/Applications/` にインストールしてください。

## 手順

1. [GitHub Releases](https://github.com/u23ken/send-control/releases) から `Send-Control-v*.zip` をダウンロードします。
2. ZIP を展開します。
3. `Send Control.app` を `/Applications/` に移動します。
4. `Send Control.app` をダブルクリックして起動します。
5. `システム設定 > プライバシーとセキュリティ > アクセシビリティ` で Send Control を許可します。
6. `システム設定 > プライバシーとセキュリティ > 入力監視` で Send Control を許可します。
7. メニューバーに Send Control のアイコンが表示されれば完了です。

## 確認方法

対象アプリで Return キーを押すと Shift+Return として入力され、Shift+Return を押すと Return として入力されます。
